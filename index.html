Web-Development


Ever seen a website and wondered how it is made? Well, web development is your answer. It is the field or domain wherein people are involved in designing websites and maintaining them for different purposes.
In the era of digitalization and the online world developing rapidly, web development is a major thing needed for sustenance. It provides a path to create websites and apps to make lives easier.

HTML Guide

(Hyper Text Markup Language) serves as the skeleton or wire-frame for the website, which is it’s basic supporting system. For creating a website, it is the first thing that is made. It is easy to learn.

CSS Guide

(Cascading Style Sheet) involves beautifying the website to make it look presentable and user friendly, appealing to the eyes. For basics, CSS is used for adding style to your HTML pages.

JavaScript Guide

Javascript (JS) serves as the “hidden operator” of a website that gets user data and gives the result. Besides this, it has various other utilities when it comes to designing websites and making them useful.



Graphic Designing


Ever seen a picture with a city video in it? Wanna know how the posters for an event are made with 3D graphics?
With the surge of increasing graphic designing, various softwares find their use. They provide different tools to enhance and edit the photos and videos and create content.

Photoshop

Photoshop provides a user friendly interface to make our work very beautiful and alluring to the eyes. It finds a major use in industries and digital imaging fields.

Illustrator

Illustrator is a software with an interface similar to photoshop with slightly different functionality. It is very handy in making logos, sketches and many visual graphics.

Blender

Every animation that you see on your screen is done with the help of Blender. It provides step wise editing of animations with minute details to produce effective visuals.
