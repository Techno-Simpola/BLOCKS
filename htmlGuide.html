HTML(Hyper Text Markup Language) is a document-layout and hyperlink-specification language.

It can basically be thought as the skeleton of a website.

It tells the browser how to display the contents of a hypertext document.

It is the first step towards learning Web Development. Once you learn HTML, you can build simple, static websites very easily.


For more information on just basics, go to this link. gfg/html
GETTING STARTED
Learning HTML can be summarized in following steps :

Learning about the basic HTML tags and attributes.

Understanding about how to incorporate the html media elements into our website.

Learning some advanced concepts like using all of your html knowledge to structure a mega project.

Tips & Tricks :
For basic HTML, work on projects like making a login page, portfolio etc.
