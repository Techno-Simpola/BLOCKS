CSS (Cascading Style Sheets) is used to simplify the process of making web pages presentable.

It is a stylesheet language used to design the webpage to make it attractive.

It can be regarded as one of the most important aspect of a good website as it increases the standards and overall look of the website which makes it easier for the user to interact.


For more information on just basics, go to this link. gfg/CSS
GETTING STARTED
Learning CSS can be summarized in following steps :

Learning about the three ways in which we can include CSS into our html pages, namely: Inline, Embedded and External CSS.

Understanding the basic tools like css selectors and how to modify the text, background ,color etc of various html elements.

Learning some more important concepts like flexbox model, grid model and animations.
